# 🌍 Brent Oil Price Analysis API 📊

Welcome to the **Brent Oil Price Analysis API**! This API provides various endpoints to analyze oil price trends, calculate yearly averages, and understand the impact of key events on oil prices.

## 🚀 **Features**

- **Price Trends**: Analyze how oil prices have changed over time.
- **Event Impact**: Understand how significant events have influenced oil prices.
- **Yearly Averages**: Calculate average oil prices for each year.
- **Price Distribution**: Get insights into the distribution of oil prices.

## 📜 **Documentation**

You can explore the full API documentation at [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) after running the application.

## 🛠️ **Requirements**

Make sure you have the following installed:

- Python 3.7 or later
- FastAPI
- Uvicorn
- Pandas

## 📦 **Installation**

Clone the repository and install the required packages:

```
git clone https://github.com/Amen-Zelealem/Brent-Oil-Price-Change-Point-Analysis.git
cd Brent-Oil-Price-Change-Point-Analysis/api
pip install -r requirements.txt
```

## 🔧 **Running the API**
To run the API, use the following command:

```
python app.py
```
